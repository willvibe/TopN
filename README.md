# 🛰️ TopN 评分系统

为各类比赛发起评分活动的 Web 应用：**后台录入比赛与作品 → 生成评分链接 → 评委扫码打分 → 大屏实时展示评委评分明细与作品排名**。

## ✨ 核心功能

- **赛控后台**：录入大赛、作品、评分标准（带权重）、评委（含等级），发布生成评分链接
- **评委评分**：每位评委专属匿名链接，移动端友好，按维度滑动打分，支持首次提交 + 1 次修改后锁定
- **实时大屏**：左侧能量柱排行榜、右侧评委评分明细（匀速滚动）、全场排名矩阵、三甲加冕
- **智能评分算法**：
  - 每位评委对作品的分 = Σ(维度分 × 维度权重)
  - 作品总分按评委**等级权重**加权平均（专家 ×1.5 / 资深 ×1.2 / 普通 ×1.0）
  - **评委数 > 5 时**自动去掉一个最高分、一个最低分，再加权平均
- **实时推送**：基于 SSE，评委提交瞬间大屏刷新

## 🧱 技术栈

Node.js · Express · MySQL · SSE 实时推送 · Tailwind CSS · GSAP

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置数据库（环境变量）

复制 `.env.example` 为 `.env` 并填入真实值，或直接在命令行设置环境变量。

**Windows (cmd)：**
```cmd
set DB_HOST=你的数据库IP
set DB_PORT=3306
set DB_USER=root
set DB_PASSWORD=你的密码
set DB_NAME=TopN
```

**Linux / macOS：**
```bash
export DB_HOST=你的数据库IP
export DB_PORT=3306
export DB_USER=root
export DB_PASSWORD=你的密码
export DB_NAME=TopN
```

### 3. 初始化数据库表

```bash
npm run init-db
```

### 4. （可选）生成样例大赛

```bash
npm run seed
```
生成「2026 全球未来科技创新大赛 · 决赛现场」：6 组作品、3 项评分标准（权重合计 100%）、7 位含等级的评委。也可在后台点击「⚡ 样例」按钮一键生成。

### 5. 启动服务

```bash
npm start
```

打开 http://localhost:5678 进入入口，或直接访问 http://localhost:5678/admin.html 开始录入。

## 📖 使用流程

1. 在**后台**创建大赛 → 录入作品 / 评分标准 / 评委 → 体检配平 → **发布**
2. 发布后获得三类链接：
   - **评委评分链接**（分享给所有评委，或用每位评委的专属链接）
   - **大屏实时排名链接**（投放到现场屏幕）
   - **后台管理链接**
3. 评委通过链接打分，大屏实时刷新排名

## 🏆 评分规则说明

- 评委**等级权重**：专家 ×1.5、资深 ×1.2、普通 ×1.0
- 评委对作品的维度加权分 = Σ(各维度分 × 维度权重)
- 作品总分：
  - 有效评委 ≤ 5 人：全部评委按等级权重加权平均
  - 有效评委 > 5 人：去掉一个最高分、一个最低分后，对其余评委按等级权重加权平均
- 大屏矩阵中，被剔除的最高/最低分会用红色标记
